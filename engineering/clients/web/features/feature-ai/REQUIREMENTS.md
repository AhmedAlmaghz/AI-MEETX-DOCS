# Web AI UI Requirements

Document ID: WEB-AI-REQ-001
Version: 1.0.0
Status: Approved

---

# 1. Purpose
Defines the client-side functional requirements for the AI Assistant panel (Q&A box, dynamic summary list).

---

# 2. Functional Requirements

- **WAI-FR-001 (Summary Tab)**: The sidebar panel SHALL render a summary log updating session descriptions in real-time.
- **WAI-FR-002 (Assistant Q&A)**: The input field SHALL allow users to post questions to the AI assistant using meeting transcripts, displaying responses as text cards.
- **WAI-FR-003 (Action Items)**: The panel SHALL compile dynamic lists of extracted action items with user assigning options.
- **WAI-FR-004 (Visual Status)**: The UI SHALL draw clean statuses notifying users when summaries compile.
