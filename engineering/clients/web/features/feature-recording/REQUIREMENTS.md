# Web Recording UI Requirements

Document ID: WEB-REC-REQ-001
Version: 1.0.0
Status: Approved

---

# 1. Purpose
Defines the functional requirements for recording indicators and playback galleries.

---

# 2. Functional Requirements

- **WRE-FR-001 (Active Badge)**: The room interface SHALL render a glowing red "REC" indicator to all participants when recording is active.
- **WRE-FR-002 (Host Toggle)**: The host toolbar SHALL render start/stop recording buttons executing platform triggers.
- **WRE-FR-003 (Playback Gallery)**: The dashboard SHALL render a list of past recordings with download and delete controls.
- **WRE-FR-004 (Format support)**: The playback player SHALL support browser native MP4 video decoding.
