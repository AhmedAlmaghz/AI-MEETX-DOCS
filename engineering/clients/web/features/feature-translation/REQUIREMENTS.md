# Web Translation UI Requirements

Document ID: WEB-TRANS-REQ-001
Version: 1.0.0
Status: Approved

---

# 1. Purpose
Defines the functional requirements for live translation subtitle rendering on the web client.

---

# 2. Functional Requirements

- **WTR-FR-001 (Subtitle Overlay)**: The meeting canvas SHALL render translated subtitle streams overlaying active speaker video grid tiles.
- **WTR-FR-002 (Target Dropdown)**: The toolbar SHALL provide dropdown list menus to select output subtitle languages.
- **WTR-FR-003 (Toggle Control)**: Participants SHALL support toggling subtitle displays on or off dynamically.
- **WTR-FR-004 (History Log)**: The sidebar panel SHALL support viewing a historical log list of running translations.
