# Web Classroom UI Requirements

Document ID: WEB-CLASS-REQ-001
Version: 1.0.0
Status: Approved

---

# 1. Purpose
Defines the client-side functional requirements for whiteboard canvas and student quiz widgets.

---

# 2. Functional Requirements

- **WCL-FR-001 (Whiteboard Canvas)**: The classroom overlay SHALL render an interactive drawing whiteboard supporting line, shape, and text annotations.
- **WCL-FR-002 (Quiz Popups)**: Students SHALL receive interactive quiz forms instantly when published by teachers, preventing room exits while quiz runs.
- **WCL-FR-003 (Breakout Staging)**: The interface SHALL display a count-down overlay routing participants into breakout rooms upon host commands.
- **WCL-FR-004 (Attendance list)**: Teachers SHALL access attendance grids displaying current engagement statistics and export options.
